# Aplicación Biblioteca con Flask

## Dependencias
Nos movemos al directorio de la aplicación.

Es mejor crear un entorno virtual: `python -m venv venv`.

Lo activamos, bien con `source venv/bin/activate`en Linux, bien con `venv\Scripts\activate`.

Instalamos las librerías necesarias con `pip install flask`

## Ejecución de la aplicación
Basta con ejecutar `python app.py` desde la línea de comandos.

Posteriormente nos conectaremos a la URL
